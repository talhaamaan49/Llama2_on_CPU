# Llama2_on_CPU


### Step 1:
### Clone the Repository

```bash
git clone https://github.com/
```

### Step 2:
Create a virtual Environment
```bash
conda create -n cpullama python==3.8 -y
```


```bash 
conda activate cpullama
```

```bash
pip install -r requirements.txt
```
